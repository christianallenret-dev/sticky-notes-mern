# Sticky Notes

A simple MERN full stack web application that allows users to create, update, and delete notes.

## Tech Stack
- MongoDB
- Express.js
- React
- Node.js

## Features
- Create and delete notes
- Has a persistent storage using MongoDB
- A RESTful API with Express.js and Node.js
- Responsive UI built with React

## Installation

```bash
# Clone the repository
git clone https://github.com/your-username/sticky-notes.git

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install

# Run the application
npm run dev
